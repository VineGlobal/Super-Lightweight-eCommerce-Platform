\Zend_Gdata_Gapps_EmailListQuery

Package: Zend_Gdata\Gapps
Assists in constructing queries for Google Apps email list entries.
Instances of this class can be provided in many places where a URL is required. For information on submitting queries to a server, see the Google Apps service class, Zend_Gdata_Gapps.
